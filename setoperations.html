set operators

In relational algebra, set operators are used to combine or manipulate
relations (tables) based on set theory concepts. Set operators allow
you to perform operations such as union, intersection, and difference
on relations. Here are the commonly used set operators in relational algebra:

  • union(U)- The union operation combines two tables, returning all distinct rows from both tables. Ex- find the names of FNAME whose CID is 1 or 2 ΠFNAME((σCID=1 CROPS) FARMERS) U ΠFNAME((σCID=2 CROPS) FARMERS)

  • Intersection(∩)- The intersection operation returns only the rows that are common to both tables. Ex- find the names of FNAME whose age is 1 and 2 ΠFNAME((σCID=1 CROPS) FARMERS) ∩ ΠFNAME((σCID=2 CROPS) FARMERS)

  • set difference(-) The difference operation returns the rows from the first table that are not present in the second table. Ex- Find the FNAME whose FID is 2 and CNAME is not CORN ΠFNAME((σFID=2 CROPS) FARMERS) - ΠFID((σCNAME=’WHEAT’ CROPS))
