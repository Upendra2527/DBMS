Rename

Rename (ρ): The rename operation changes the name of attributes or relations
in a table. It is denoted by the ρ symbol.
Example: we have a table called "FARMER" with columns: "FNAME",
"AGE" and "CROPS." If we want to rename the "FNAME" column to "FARMER_NAME"
we can use the rename operation as follows:

ρ(FARMER_NAME/FNAME)(FARMER)
